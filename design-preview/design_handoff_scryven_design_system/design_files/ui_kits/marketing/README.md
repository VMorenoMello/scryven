# Marketing UI kit

A single-page landing for scryven.com.br. Same tokens as the web app — the brand reads consistently across surfaces.

Sections (top to bottom):
- Nav with logo + auth actions
- Hero: pull-quote headline + capture-style entry + product peek
- The matrix explained: 2×2 visual + short copy per quadrant
- "Three a day" — daily cadence pitch
- Pricing — single tier, simple
- Footer

## Files
- `index.html` — the page
- `globals.css` — shared tokens (mirrored from root)

## Disclaimer
No production marketing site exists yet — this is the design system's interpretation of the brief. The hero peek uses the web-app components verbatim so visuals stay 1:1.
